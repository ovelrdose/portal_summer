1. Общая цель
Реализовать возможность для преподавателя просматривать, оценивать и комментировать прикреплённые студентами домашние задания, а также обеспечить студентам возможность просматривать полученные оценки и комментарии.

2. Требования к функционалу
2.1. Страница управления домашними заданиями (для преподавателя)
На странице редактирования курса (или просмотра курса преподавателем) добавить элемент (иконка / текст / бейдж), отображающий количество прикреплённых домашних заданий по каждому разделу (модулю) курса.
По клику на этот элемент — переход на отдельную страницу «Проверка домашних заданий».
На этой странице:
Отображается список разделов, в которых есть хотя бы одно прикреплённое ДЗ.
Для каждого раздела — список студентов, прикрепивших ДЗ, с указанием:
ФИО студента;
Дата прикрепления;
Ссылка/кнопка «Скачать файл(ы)».
Для каждого прикреплённого ДЗ предусмотрены поля:
Текстовое поле для комментария преподавателя (блок редактора, аналогичный используемому при наполнении курса — например, Rich Text Editor);
Поле для оценки (числовое значение или выпадающий список — уточнить в дальнейшем, пока — текстовое поле с валидацией на число).
Кнопка «Сохранить оценку» — сохраняет комментарий и оценку в БД.
2.2. Отображение результата для студента
На странице прохождения курса, в блоке прикрепления домашнего задания (под формой загрузки), если ДЗ уже проверено:
Отображается оценка (только если она задана);
Отображается комментарий преподавателя (в виде обычного текстового блока, стилизованного под «ответ преподавателя»).
Если оценка и комментарий отсутствуют — ничего не отображается.
