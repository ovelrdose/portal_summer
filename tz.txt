Я хочу переработать свою админ панель. Там сейчас есть некоторая статистика, модуль управления пользователями и модуль управления контентом.
Я хочу на странице "Админ панель" оставить только  модуль управления пользователями и модуль управления контентом. Остальное я хочу перенести в другую страницу, которая будет называться "Статистика".
Странице "Статистика" будет доступна из того же места, что и админ панель (при нажатии на имя пользователя), но должна быть доступа не только администратору, но и преподавателям
Сама эта страница должна иметь следующий вид:
. Раздел «Глобальная статистика» (доступно Администратору)
Раздел представляет собой дашборд с выводом ключевых метрик и таблиц.

Блок общих метрик (цифры):

Посещаемость (за период):

Уникальные посетители.

Количество просмотров страниц.

Количество пользователей:

Общее число зарегистрированных.

Новые пользователи за выбранный период.

Количество курсов:

Общее количество активных курсов.

Количество курсов, созданных за период.

Блок рейтингов:

Топ-10 активных пользователей:

Критерий: наибольшее количество входов в систему или наибольшее количество выполненных действий за период.

Вывод: Имя пользователя, Email, Количество действий/входов.

Топ-10 популярных курсов:

Критерий: наибольшее количество подписчиков ( enrollment) или наибольшее количество просмотров материалов.

Вывод: Название курса, Автор, Количество подписчиков.

 Блок выгрузок (таблицы):

Выгрузка пользователей по ролям:

Выпадающий список для выбора роли (Студент, Преподаватель, Админ и т.д.).

Отображение таблицы пользователей выбранной роли (с пагинацией).

Колонки: ID, Имя, Email, Дата регистрации, Последний вход.

Кнопка «Экспорт в CSV» для выгруженной таблицы.

Раздел «Статистика в рамках курса» (доступно Администратору и Преподавателю)
Интерфейс должен позволять выбрать курс и получить по нему детальную статистику.

Выбор курса:

Выпадающий список (select) с курсами пользователя.

Для администратора: список всех курсов платформы.

Для преподавателя: список только его курсов.

При смене значения в списке, вся статистика ниже должна динамически перезагружаться

Блок статистики по выбранному курсу:
После выбора курса отображаются следующие метрики (за весь период существования курса или с возможностью выбора дат):

Количество подписчиков: Общее число студентов на курсе.

Количество выполненных ДЗ: Сколько домашних заданий отмечено как "Выполнено/Проверено".

Количество просроченных ДЗ: Количество выданных заданий, у которых дедлайн истек, но статус не "Выполнено".

Распределение оценок: Средняя оценка, гистограмма (пятерки, четверки и т.д.) или просто статистика по количеству отличников/двоечников.

2.3.3. Экспорт по курсу:

Для каждого блока статистики (или для всей страницы целиком) предусмотреть кнопку «Экспорт данных курса в CSV». В файл должна попадать полная информация: название курса, выбранный период (если есть), все цифровые метрики.
