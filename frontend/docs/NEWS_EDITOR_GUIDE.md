# Руководство по редактору новостей

## Введение

Редактор новостей позволяет создавать и публиковать новости на образовательном портале. Новости могут содержать текст, изображения и ссылки.

## Доступ к редактору

1. Войдите в систему с правами администратора
2. Перейдите в раздел "Новости" в меню
3. Нажмите кнопку "Создать новость" или выберите существующую новость для редактирования

## Основные элементы редактора

### 1. Заголовок
- Обязательное поле
- Отображается в списке новостей и на странице новости

### 2. Краткое описание
- Обязательное поле
- Максимум 500 символов
- Отображается в превью новости в списке

### 3. Обложка новости
- Необязательное поле
- Поддерживаемые форматы: JPEG, PNG, GIF, WebP
- Максимальный размер: 5 МБ
- Рекомендуемое разрешение: 1200x630 пикселей

### 4. Теги
- Позволяют категоризировать новости
- Можно выбрать несколько тегов
- Нажмите на тег для его выбора/отмены

**Доступные теги:**
- События - для анонсов мероприятий
- Образование - для учебных материалов
- Объявления - для важных сообщений
- Новости курса - для обновлений по курсам
- Достижения - для успехов учащихся
- Расписание - для изменений в расписании

### 5. Содержимое новости (блочный редактор)

Редактор поддерживает следующие типы блоков:

#### Текстовый блок
- Для добавления текста с форматированием
- Поддерживает жирный, курсив, зачеркнутый текст
- Поддерживает заголовки и списки

#### Блок изображения
- Для вставки изображений в текст новости
- Поддерживает подпись к изображению
- Максимальный размер: 5 МБ

#### Блок ссылки
- Для добавления внешних ссылок
- Можно указать название ссылки и URL

## Рабочий процесс

### Создание новой новости

1. Нажмите "Создать новость"
2. Заполните заголовок и краткое описание
3. Выберите обложку (опционально)
4. Выберите теги
5. Добавьте содержимое с помощью блочного редактора
6. Выберите действие:
   - **Сохранить в черновик** - сохранить без публикации
   - **Опубликовать** - сохранить и опубликовать новость

### Редактирование существующей новости

#### Редактирование черновика:
- Внесите изменения
- Нажмите **"Сохранить в черновик"** для сохранения без публикации
- Нажмите **"Опубликовать"** для публикации

#### Редактирование опубликованной новости:
- Внесите изменения
- Нажмите **"Сохранить"** для сохранения изменений
- Нажмите **"Вернуть в черновик"** для снятия с публикации

### Удаление новости

1. Откройте новость для редактирования
2. Нажмите кнопку "Удалить новость" (внизу справа)
3. Подтвердите удаление

**ВНИМАНИЕ:** Удаление необратимо!

## Статусы новостей

### Черновик
- Новость не отображается в общем списке
- Доступна только администраторам
- Можно редактировать без ограничений

### Опубликована
- Новость видна всем пользователям
- Отображается в списке новостей
- Можно редактировать и сохранять изменения
- Можно вернуть в черновик

## Полезные советы

1. **Используйте краткое описание** - оно показывается в превью новости, делайте его привлекательным

2. **Добавляйте обложку** - новости с изображениями привлекают больше внимания

3. **Выбирайте правильные теги** - они помогают пользователям находить нужные новости

4. **Сохраняйте черновики** - не торопитесь с публикацией, сохраните черновик и вернитесь к нему позже

5. **Проверяйте перед публикацией** - убедитесь, что все изображения загружены и ссылки работают

6. **Используйте изображения оптимального размера** - большие файлы замедляют загрузку страницы

## Решение проблем

### Теги не отображаются
**Решение:** Обратитесь к администратору для создания тегов в системе.

### Не удается загрузить изображение
**Возможные причины:**
- Файл слишком большой (>5 МБ)
- Неподдерживаемый формат файла
- Проблемы с подключением к серверу

### Изменения не сохраняются
**Решение:**
- Проверьте, заполнены ли все обязательные поля
- Проверьте консоль браузера на наличие ошибок (F12)
- Обратитесь к администратору

## Горячие клавиши

В текстовом редакторе:
- **Ctrl/Cmd + B** - жирный текст
- **Ctrl/Cmd + I** - курсив
- **Ctrl/Cmd + S** - сохранить (работает в некоторых браузерах)
